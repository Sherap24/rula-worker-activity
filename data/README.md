# Data directory (repository-local)

Raw construction datasets **must not** be stored here. The project lives in OneDrive;
large files belong under `RULA_DATA_ROOT` (see `docs/DATA_ACQUISITION.md`).

This folder holds **repository-local** processed outputs only:

- `processed/` — derived features, pose CSVs (Week 3+), gitignored
- `manifests/` — clip inventory and split manifests generated by CLI tools

Generated manifests are gitignored except this README.
